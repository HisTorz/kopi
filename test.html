📱 Test Responsivitas

Untuk test responsivitas:

  1. Buka website di browser
  2. Tekan F12 untuk Developer Tools
  3. Klik icon mobile/tablet untuk test berbagai ukuran layar
  4. Test menu mobile dengan hamburger button

🎨 Customization

Untuk mengganti gambar placeholder:

  1. Tambahkan foto asli ke folder images/
  2. Update HTML untuk menggunakan gambar asli
  3. Lihat images/README.md untuk panduan lengkap

🐛 Troubleshooting

Jika ada masalah:

  • Pastikan semua file ada dalam satu folder
  • Check console browser untuk error JavaScript
  • Pastikan koneksi internet untuk Google Fonts dan Font Awesome
  • Test di browser yang berbeda
